# Groww App Review Weekly Pulse

This project converts recent Groww app reviews into a weekly product pulse.

It was built for **NextLeap LIP Challenge 3**.

---

# Workflow

1. Import recent App Store and Play Store reviews
2. Group reviews into themes
3. Generate a weekly one-page product pulse
4. Draft a summary email

---

# Files

reviews.csv → sample dataset of recent reviews

weekly_pulse.md → one-page weekly product note

email_draft.md → draft email summary

---

# Theme Legend

Onboarding / KYC  
Customer Support  
App Performance  
Withdrawals / Payments  
Ease of Use

---

# How to Re-run for a New Week

1. Export the latest app reviews
2. Replace the data in reviews.csv
3. Regenerate the weekly note following the same structure

---

# Notes

• Reviews are from public sources only  
• No personal identifiable information is included  
• Quotes are anonymized
